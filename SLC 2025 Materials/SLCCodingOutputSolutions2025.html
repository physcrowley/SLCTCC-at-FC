SLC CODING COMPETITION 2025

Judging a problem:

  • Write down the time and the problem completed (identified by the number), then provide the team with the proper input file as indicated under each problem’s expected output.
  • For every problem, each line of correct output is worth ten points. The score for fully correct output is 100 points.
  • If a team completes a problem with all output lines correct on their first attempt, they receive ten bonus points.
  • Teams may only run their program once per attempt. If a team wishes to change their code and try again, they will need to use their second attempt.
  • After a team’s first attempt, they may keep the input file from their first attempt and may request to write down the expected output of the first attempt.
  • Note that in order for a program’s output to be worth points, it must match the format of the expected output.
