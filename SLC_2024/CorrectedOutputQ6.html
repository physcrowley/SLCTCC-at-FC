SLC CODING COMPETITION 2024

Judging a problem:

  • Write down the time and the problem completed, then provide the team with the proper input file as indicated under each problem’s expected output.
  • For every problem except #6, each line of correct output is worth ten points. The score for fully correct output is 100 points on each question.
  • Each line of output is worth five points on #6 as there are twenty lines of output instead of ten.
  • If a team completes a problem with all output lines correct on their first attempt, they receive ten bonus points.
  • If teams wish to change their code and try again, they will have to do so on their second attempt.
  • If a team wishes for a second attempt, they may keep the input file from the first attempt and they may request to write down the expected output from the first attempt.
